Создать базу данных для анализа статистики игры Dota 2, используя графовые возможности SQL Server. База должна хранить информацию о:
Профессиональных игроках
Героях игры
Игровых предметах
Связях между ними (игроки → герои, герои → предметы, игроки → предметы)
1) Найти всех игроков, которые играют на конкретном герое (например, Invoker)
2) Найти все предметы, которые предпочитает конкретный игрок (например, Miracle-)
3) Найти топ-2 самых популярных предметов для конкретного героя (например, Anti-Mage)
4) Найти топ-5 героев по количеству сыгранных матчей
5) Найти все связи между игроком, его героями и предпочитаемыми предметами (например, для Arteezy)
